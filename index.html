Assignment 1 & 2

One of the biggest coding project that I have taken up has to be the Assignment 1 and 2 in COMP 2511. Bir (Partner) and I have designed a car website from my inspiration from cars. The website included three pages with three cars and more of the content on the website was generated from ChatGPT.

Although, generating the content was fairly easy and sketching was fairly easy, the HTML5 and CSS part was much more difficult. With minimum knowledge about CSS, it was super complicated to position the images around the website so they stay relative to other things in the page.

Some other challanges came in during assignment 2 where I was suppose to make the existing website responsive. Except for the content, none of the code was close to salvageble. We had to start the whole project again but using grids and media quries to make it mroe responsive.

Reflection: While doing the responsive part of the website, I have realized that it is not easy to make a website responsive. From here on out, I am planning on using grid and relative fonts to design websites. Using grids, it will be much easier to make a webiste responsive without much more work.

Java - Truck Sim

One of the most interesting projects that I have taken up was a truck simulator for my Gr 11 Java class. I had the freedom to build any game I would like and it didnt have to have any graphics. Although, I wanted to make it graphicial I reliazed that I did not have the skills to do that.

The truck sim came along really well except for the fact that I could not add all the features that I wanted to. The main goal of the truck sim was the user owns a logistics company that they have to build into an empire. The game included choices for the materials the user wanted to transport and how they would like to transport. The user had the choice to do it the right way but they could also speed which could potentailly get them speeding tickets.

The one thing I learned from that project is how critical time management was. I did not have the best plan for the tackle the things I wanted. The time constraints did not allow me to add some features.
